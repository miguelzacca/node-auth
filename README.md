# Auth with MySQL

Simple authentication with Nodejs, MySQL, Sequelize, and JWT.

## Install

Dependencies:

- `sequelize`
- `mysql2`
- `zod`
- `bcrypt`
- `express`
- `dotenv`
- `jsonwebtoken`

```
npm install
```

## Use

```
node src/index.js
```

or for development (nodemon)

```
npm run dev
```
