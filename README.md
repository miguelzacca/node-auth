# Authentication with MySQL

Simple authentication with Nodejs, MySQL, Sequelize, and JWT.

## Install

Dependencies:

- `sequelize`
- `mysql2`
- `cors`
- `zod`
- `xss`
- `bcrypt`
- `express`
- `dotenv`
- `fs`
- `jsonwebtoken`

```bash
npm ci
```

## Use

```bash
npm run server
```

or for development (nodemon)

```bash
npm run dev
```
